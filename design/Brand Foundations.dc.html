1 · Logo

Wordmark set in Source Serif 4 Semibold, ending in a full point set in seal red. A mandate is an instruction given and accepted — the point is that decision, made. Quietly confident, legible everywhere, and the only place the accent colour appears. The mark is the initial "M." on the ink tile.

2 · Colour

Ink blue on warm paper, with one seal-red accent reserved for the wordmark's full point — never for status meaning. All pairs below pass WCAG 2.1 AA for their usage size. Statuses, confidence and claim types always carry an icon/shape and full text label — colour is never the only signal.
